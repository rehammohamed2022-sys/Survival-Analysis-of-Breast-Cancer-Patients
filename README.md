Breast Cancer Survival Analysis

Survival Analysis Project

Faculty of Economics and Political Science, Cairo University
Overview

This project applies survival analysis techniques to investigate factors influencing survival among breast cancer patients using the Rotterdam Breast Cancer Dataset.

The study evaluates survival patterns, identifies significant prognostic factors, and examines how clinical and pathological characteristics affect patient outcomes over time.

 Research Objectives

- Estimate overall survival probabilities among breast cancer patients.
- Compare survival experiences across different patient groups.
- Identify significant predictors of mortality risk.
- Evaluate the proportional hazards assumption.
- Develop an extended Cox model to account for time-varying effects.

Statistical Methods

- Exploratory Data Analysis (EDA)
- Kaplan–Meier Survival Analysis
- Log-Rank Test
- Cox Proportional Hazards Regression
- Schoenfeld Residual Analysis
- Extended Cox Regression with Time-Dependent Covariates
- Forest Plot Visualization

 Software

- R
- RStudio

Dataset

Rotterdam Breast Cancer Dataset

Number of Patients: 2,982 

Key Findings

- Median survival time was approximately 4,033 days (about 11 years). 
- Tumor size and lymph node involvement were among the strongest predictors of survival.
- Larger tumors were associated with substantially higher mortality risk. :contentReference
- Higher lymph node involvement significantly increased the hazard of death. :contentReference
- Grade 3 tumors were associated with worse survival outcomes compared to Grade 2 tumors.
- Violations of the proportional hazards assumption were addressed using an Extended Cox Model with time-dependent covariates.
